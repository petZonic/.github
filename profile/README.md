# PetZonic 🐾

**India's Premier Pet Marketplace & Services Platform**

We connect pet lovers with trusted breeders, sellers, and service providers — offering a comprehensive ecosystem for pet commerce, care, and community.

## Our Repositories

| Repository | Description |
|------------|-------------|
| [petzonic-api](https://github.com/petzonic/petzonic-api) | Backend API (NestJS + PostgreSQL) |
| [petzonic-web](https://github.com/petzonic/petzonic-web) | Customer website (Next.js) |
| [petzonic-customer-app](https://github.com/petzonic/petzonic-customer-app) | Customer mobile app (Flutter) |
| [petzonic-seller-app](https://github.com/petzonic/petzonic-seller-app) | Seller mobile app (Flutter) |
| [petzonic-infra](https://github.com/petzonic/petzonic-infra) | Infrastructure & DevOps (Docker, Terraform, CI/CD) |
