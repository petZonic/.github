# .github

Organization-wide community health files for **@petzonic**.

Files in this repo are automatically inherited by all repositories in the `petzonic` organization that don't define their own.

## What's included

| File | Purpose |
|------|---------|
| `.github/ISSUE_TEMPLATE/bug_report.md` | Bug report template for all repos |
| `.github/ISSUE_TEMPLATE/feature_request.md` | Feature request template for all repos |
| `.github/pull_request_template.md` | Default PR checklist |
| `CONTRIBUTING.md` | Contribution guidelines |
| `CODE_OF_CONDUCT.md` | Community code of conduct |
| `CODEOWNERS` | Default code ownership rules |
| `profile/README.md` | Organization profile (shown on github.com/petzonic) |

## Override per repo

Any repo can override these defaults by placing its own version of the file in its own `.github/` directory.
